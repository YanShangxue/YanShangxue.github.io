3、java.lang.String

3.1、说明

  • 说明一：自动默认导入java.lang.*中的类（不包括子包）

  • 说明二：字符串的本周是字符数组char[]

    char[] arr ={'张','三'}

  • 说明三：字符串类型是Java所有类型中最常用的

3.2、常用方法

String s ="abcde";

  1. equals方法：判断字符串是否相等

    boolean a=s.equals("abc");

  2. charAt方法：根据下标返回字符

    char b =s.charAt(2);

  3. indexOf方法：查找指定字符串、找到返回起始下标

    int c =s.indexOf("abc");

  4. length方法：字符串长度

    int d =s.length();

  5. startsWith方法：判断是否以某字符串开头

    bool e=s.startWith("ab");

  6. contains方法：是否包含某字符串

    bool f =s.contains("bc");

  7. substring方法：截取

    string g=s.substring(2); //截取下标为2之前的字符串

    string h=s.substring(1,3); //截取下标为1-2的字符串

  8. replace方法：替换指定的字符串

    String i =s.replace("bc","123");

  9. trim方法：去除两边空格

    String j=" abc ".trim();

  10. toUpperCase方法：转换大写

    String k="abc".toUpperCase();

3.3、字符串创建方式

方式一：字面值、建议

  • 说明：String开发中常用，Java对字面值进行了优化

  • 原理：先检查内存中是否有此字符串、有责直接返回、如果没有再创建新字符串。

  • 示例：String s1 ="abc";

    String s2=new String("你好")；

    String s3=new String("你好")；

    System.out.println(s2 == s3); //ture

    System.out.println(s2.equals(s3)); //ture

方式二：new创建对象、不建议

  • 说明:new每次都会创建新的字符串
  • 类类型：==比较的是地址、equaks比较真实的值

String s2=new String("你好")；

String s3=new String("你好")；

System.out.println(s2 == s3); //false

System.out.println(s2.equals(s3)); //ture

3.4、补充内容

  1. String字符串对象一旦创建就不能更改
  2. 修改字符串的方法都是放回一个新的字符串
  3. "+"在字符串中是连接字符串的意思、并可以自动把其他类型转换为字符串再连接
  4. 由于字符串对象不能改变、所以如果频繁修改字符串，不建议使用String类型（用StringBuilder）
  5. append 方法：用来连接字符串（类似+），但是不创建新对象，不会改变字符串地址。
